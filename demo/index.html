PoC for resizable elements

This page shows how resizable elements work in practice.
The resizable elements are controlled by the resize attribute which contains a list of values that indicate in which direction the element can be resized. Each value describe a geographical direction like n for North, e for East, and so forth.
The interaction is bases on changing the cursor to "resize" on the selected direction of the resize action. The the user can move the edge to the desired position. Once the resize action finishes, the element dispatches the resize event which authors can handle to persist user configuration.
